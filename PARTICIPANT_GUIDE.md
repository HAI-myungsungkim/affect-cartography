# 참가자 안내문 (피험자용) — Affect Cartography Pilot

> 이 문서는 IRB 승인 후 피험자에게 배포할 수 있는 안내문 템플릿입니다.
> 연구진이 실제 모집 시점에 맞춰 일부 항목(연구 기간, 보상, 연락처)을 수정해 사용하세요.

---

## 안녕하세요! 연구 참여에 감사드립니다.

본 연구는 **하루 약 10분, 4주간** 자신의 감정을 더 세밀하게 인식하는 훈련을 받는 프로그램입니다.

## 1. 앱 설치 방법

### Android 사용자
1. 연구진이 보낸 **Google Play 내부 테스트 초대 링크**를 클릭
2. "테스터 되기" 동의 → Play Store에서 앱 다운로드
3. 또는 연구진이 제공한 APK 파일을 직접 설치 (설정 → 알 수 없는 출처 허용)

### iOS 사용자
1. 앱스토어에서 **TestFlight** 앱 먼저 설치
2. 연구진이 보낸 **TestFlight 초대 링크** 클릭
3. TestFlight에서 "Affect Cartography" 설치

## 2. 첫 로그인

1. 앱 실행
2. 연구진이 발급한 **피험자 코드(예: P003)** 입력
3. **본인 실명** 입력
4. "로그인" 버튼 → 첫 로그인 시 이 스마트폰이 자동 등록됩니다

⚠️ **중요**: 이후 다른 스마트폰에서는 로그인할 수 없습니다. 휴대폰을 변경하실 일이 생기면 **연구진에게 미리 연락**해 주세요.

## 3. 알림 받기

- 앱 첫 실행 시 알림 권한을 허용해 주세요
- 매일 3번 알림이 옵니다:
  - **오전** 9:00~12:00 중 무작위
  - **오후** 13:00~17:00 중 무작위
  - **저녁** 19:00~22:00 중 무작위
- 알림 시간대는 햄버거 메뉴 → "알림 설정"에서 조정 가능합니다
- 15분 내에 응답이 없으면 한 번 더 알림이 옵니다

## 4. 한 번 응답하는 데 약 2~5분

각 응답은 다음 흐름으로 진행됩니다:

1. **정동 기록** (10초): 화면의 격자에서 지금 기분의 위치를 점으로 찍거나, 손가락을 끌어 최근 3시간의 변화를 그리기
2. **AI와 짧은 대화** (1~2분): AI가 묻는 2~4개 질문에 자유롭게 답하기
3. **감정 단어 선택** (30초): 추천된 5개 단어 중 가장 가까운 것 선택, 필요시 인접 단어로 더 탐색
4. **강도 표시** (5초): 그 감정이 얼마나 강한지 1~5점
5. **짧은 실천 제안** (30초): 자기거리두기 / 호흡 / 작은 행동 중 하나의 짧은 안내문 읽기

## 5. 대시보드

- 홈 화면의 "대시보드 보기" 버튼으로 자신의 4주간 기록을 확인할 수 있습니다
- 정동 궤적 + 감정 단어 기록 + 응답률이 표시됩니다

## 7. 데이터 사용

- 모든 데이터는 암호화되어 국내(NCP) 서버에 저장됩니다
- OpenAI 대화는 학습에 사용되지 않습니다 (Zero Data Retention)
- 다운로드되는 분석 데이터에서 실명은 마스킹됩니다 (예: 김철수 → 김○○)
